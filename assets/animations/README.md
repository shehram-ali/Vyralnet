# Lottie Animations

This directory contains Lottie animation JSON files used in the app.

## Required Files:

### 1. loading.json
- **Source**: https://lottiefiles.com/free-animation/loading-lottie-animation-7xyTDwuIwc
- **Usage**: Loading spinner (displayed in green color)
- **Location**: Place in this directory as `loading.json`

### 2. checkmark.json
- **Source**: https://lottiefiles.com/free-animation/checkmark-DkofmnGgNy
- **Usage**: Success modal checkmark animation
- **Location**: Place in this directory as `checkmark.json`

## How to Download:
1. Visit the links above
2. Click "Download" button
3. Select "Lottie JSON" format
4. Rename files to match the names above
5. Place them in this directory
